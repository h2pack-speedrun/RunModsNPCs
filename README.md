# RunModsNPCs

> NPC spawn and routing modifiers for speedrun-focused run shaping.

Part of the [Speedrun modpack](https://github.com/h2pack-speedrun/speedrun-modpack).

## What It Does

RunModsNPCs changes a small set of NPC and route-adjacent behaviors that matter for attempt consistency.

The current module includes:

- `Disable Arachne Pity`
  Turns off Arachne pity.
- `Disable Selene Before First Boon`
  Prevents Selene from appearing before the first boon or hammer is obtained.
- `Force Arachne Spawn`
  Forces Arachne to appear.
- `Force Medea Spawn`
  Forces Medea to appear.
- `Prevent Echo Scam`
  Blocks both Fields minibosses from spawning at room 3 so Echo cannot be crowded out there.

Use this module when you want tighter control over NPC-related route variance without changing the broader world structure.

## Installation

Install via [r2modman](https://thunderstore.io/c/hades-ii/) or manually place in your `ReturnOfModding/plugins` folder.

This module is usually installed as part of the full [Speedrun modpack](https://github.com/h2pack-speedrun/speedrun-modpack), where it appears in the shared Speedrun UI with the other speedrun-focused modules.
